**Author**: Rick Hayes

## Description
Forensic tool to extract and log file contents, styled for NSFW evidence recovery.

## Features
- Recursive directory scanning
- Error-tolerant file reading
- Custom output file

## Requirements
- Python 3.x

## Usage
```bash
python3 naughty_data_dumper.py --dir /path/to/evidence --out dump.txt
```
